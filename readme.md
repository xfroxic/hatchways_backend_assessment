# Hatchways Back-End Technical Assessment
```
An application built for a back-end technical assessment provided by Hatchways.
The application fetches data from a given api based off of user input.
The user enters the given URL followed by the query parameters.
```

### Instructions
```
1. Open the project directory and enter 'npm i' in a new terminal window
2. Enter 'npm run devStart' to start the node server
3. Route 1 solution will be found at http://localhost:3000/api/ping
4. Route 2 solution will be found by entering a tag (required) and sort (optional) parameters 
   at the end of http://localhost:3000/api/posts/
```

### Examples
```
http://localhost:3000/api/posts/design
http://localhost:3000/api/posts/design,culture
http://localhost:3000/api/posts/design,culture/reads/asc
```
